# 🏨 Hotel Management System

A JavaFX-based desktop application that simulates real-world hotel operations including booking lifecycle, housekeeping workflow, and customer feedback system.
---


## 🚀 Features

### 🛏️ Room Management

* Add, update, and delete rooms
* View room availability in real-time
* Categorize rooms (Single, Double, Deluxe, etc.)

### 📅 Booking System

* Create new bookings with customer details
* Automatic room allocation based on availability
* Track booking history

### ❌ Booking Cancellation

* Cancel bookings easily
* Update room availability instantly after cancellation

### 🗑️ Record Deletion

* Delete customer or booking records
* Maintain clean and updated system data

### ⭐ Reviews & Feedback

* Customers can leave reviews for their stay
* Store and display feedback for service improvement

### 🧹 Housekeeping Management

* Track room cleaning status
* Mark rooms as cleaned / pending
* Ensure rooms are ready before new bookings

### 💵 Billing System

* Automatic calculation of total bill based on stay duration
* Transparent pricing system

### 👤 Customer Management

* Store customer details securely
* View and manage customer records

### 🧾 Modular OOP Design

* Clean separation using classes like Customer, Room, Booking
* Easy to maintain and extend
